# sapho-site

Public distribution site for **sapho** — the plain-files product source of truth.

- `index.html` — marketing page (human face) with embedded agent setup spec
- `install.md` — capability-detecting installation spec (agent face)
- `llms.txt` — agent discovery pointer

Deployed on Vercel. The install sentence on the page self-assembles from the deployed URL.
The private corpus (actual truth files) lives separately in `damianr/sapho`; this repo
contains conventions only — no user data.
